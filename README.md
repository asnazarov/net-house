### Развёртывание проекта:
   * Клонировать репозиторий в терминал командой: ```git clone https://github.com/asnazarov/net-house.git```
   * Установка зависмостей: ```npm i```
   * Запуск тестовой сборки на локальном сервере: ```npm start```
